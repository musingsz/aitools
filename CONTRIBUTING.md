# 🤝 贡献指南

感谢您对 AI 工具集开源项目的关注！我们欢迎各种形式的贡献。

## 📋 贡献方式

### 1. 提交新工具

如果您发现了新的 AI 工具并希望收录到本项目中，请按照以下格式提交：

**在对应的分类 Markdown 文件中添加：**

```markdown
| **工具名称** | 工具简介 | [访问 →](https://ailnks.com/tool/xxxx) |
```

### 2. 提交 Issue

如果您有以下需求，请通过 [GitHub Issues](https://github.com/your-org/aitools/issues) 提交：

- 🐛 发现分类错误或链接失效
- 💡 建议新增分类
- ✨ 功能建议
- ❓ 问题咨询

### 3. 提交 Pull Request

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

## 📝 提交规范

### 新增工具格式

为了保证项目的一致性，请严格遵循以下格式：

```markdown
| **工具名称** | 一句话简介（不超过 30 字） | [访问 →](https://ailnks.com/tool/xxxx) |
```

### Commit 规范

请使用语义化的 Commit Message：

- `feat: 新增 xxx 工具`
- `fix: 修复 xxx 链接`
- `docs: 更新 xxx 文档`
- `chore: 维护性更新`
